# Mon CV Interactif

## Description

Ce projet est un CV en ligne interactif réalisé dans le cadre du module de Développement Web (Licence 2 Informatique, Université Iba Der Thiam de Thiès).

## Technologies utilisées

- **HTML5** : Structure sémantique
- **CSS3** : Design moderne, responsive, animations
- **JavaScript (Vanilla JS)** : Interactivité, DOM manipulation
- **Google Fonts** : Typographie (Roboto)
- **Font Awesome** : Icônes

## Fonctionnalités implémentées

- ✅ Mode sombre / clair avec sauvegarde localStorage
- ✅ Menu hamburger responsive
- ✅ Barres de compétences animées au scroll
- ✅ Formulaire de contact avec validation client
- ✅ Bouton retour en haut
- ✅ Navigation fluide (smooth scroll)
- ✅ Design responsive (mobile-first)
- ✅ Projets avec statuts et liens

## Installation

1. Télécharger ou cloner le dossier
2. Ouvrir `index.html` dans un navigateur (Chrome, Firefox, Edge)
